PROTOCOL MESSAGE // STEP 03

mining.notify

Server push message containing work parameters for the miner

Job Distribution

Pushed by the pool to the miner. Contains all fields to build the block header and coinbase transaction â€” except extranonce2, which the miner generates.

Parameters Order

1. job_id ID used when submitting shares.
2. prevhash Hash of the previous block (LE).
3. coinb1 Initial part of coinbase tx.
4. coinb2 Final part of coinbase tx.
5. merkle_branch List of hashes for merkle root calc.
6. version Block version number.
7. nbits Encoded network difficulty.
8. ntime Current block timestamp.
9. clean_jobs If true, drop previous jobs immediately.
Server Push Notification 
{
  "id": null,
  "method": "mining.notify",
  "params": [
    "bf",             // job_id
    "4d16b6f85af6...", // prevhash
    "010000000100...", // coinb1
    "f85af6...0000",   // coinb2
    [                 // merkle_branch
      "f85af6...",
      "000000..."
    ],
    "00000002",       // version
    "1c2ac4af",       // nbits
    "504e86b9",       // ntime
    true              // clean_jobs
  ]
}

// Miner Action:

1. Clean previous jobs (if true)
2. Generate Extranonce2
3. Construct Coinbase:
   coinb1 + Ex1 + Ex2 + coinb2
4. Build Merkle Root
5. Hash Block Header
