Architecture Overview

Transport Layer

Plain TCP connection, typically on port 3333. Uses line-delimited JSON-RPC messages.

Roles & Directionality

Pool: Pushes jobs (mining.notify)
Miner: Submits shares (mining.submit)

Security Model (None)

No encryption (TLS optional/rare).
No server authentication (MITM possible).
"Trust on first use" is the default behavior.
